# Easy hooks TS with React

This example shows how to use Easy hooks TS with [React](https://reactjs.org).

## Instructions

1. Install Easy hooks TS via npm:

```
npm install
```

2. Run React script locally:

```
npm start
```

3. Open URL in the browser:

```
http://localhost:3000
```
